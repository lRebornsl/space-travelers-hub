# 📖 React To-Do List <a name="about-project"></a>

**Space Travelers' Hub** is an application created with REACT that provides commercial and scientific space travel services. The application will allow users to book rockets and join selected space missions.

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Show rockets and missions**
- **Book rockets**
- **Join and cancel missions**
- **Show missions stats**
- **Show rockets stats**
